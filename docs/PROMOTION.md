## 📝 产品介绍

### 一句话介绍

```
灵笼看板 - 为中小型团队打造的现代化组织架构可视化管理工具
```

### 三句话介绍

```
灵笼看板是一个基于 Lit + TypeScript 的开源团队管理工具。
它提供组织架构可视化、任务看板和实时系统监控三大核心功能。
专为19人左右的中小型团队设计，帮助管理者实时掌握团队动态。
```

### 详细描述

```
灵笼看板是灵笼集团开源的一款现代化团队管理工具，采用 Web Components 
架构，基于 Lit 框架和 TypeScript 开发。主要功能包括：

1. 组织架构可视化：以卡片式布局展示团队成员结构，支持按部门筛选，
   实时显示成员在线状态。

2. 任务看板：类似 Trello 的四列看板（待办/进行中/审核中/已完成），
   支持优先级标签、负责人分配和拖拽排序。

3. 实时监控：通过 WebSocket 连接实时显示 Gateway、数据库、Agent 
   等系统组件的健康状态。

技术栈：Lit 3.1.0 + TypeScript 5.3 + Vite 5.0
特点：响应式设计、类型安全、易于扩展、部署简单
```
