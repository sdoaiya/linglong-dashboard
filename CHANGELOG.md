## [1.0.0] - 2026-04-01

### 初始版本发布 🎉

#### 新增
- 组织架构可视化模块
  - 6个部门卡片展示
  - 成员列表和状态显示
  - 部门统计信息
- 任务看板模块
  - 四列看板布局
  - 优先级标签
  - 任务统计
- 实时监控模块
  - 系统健康状态
  - Agent 状态列表
  - 在线用户统计
- WebSocket 实时通信
  - 自动重连机制
  - 心跳检测
- 响应式 UI 设计
  - 深色主题
  - CSS 变量系统
  - 移动端适配

### 技术实现
- Lit 3.1.0 Web Components 框架
- TypeScript 5.3 类型安全
- Vite 5.0 构建工具
- WebSocket 实时通信
